# ALP-B Petcare Project

## Backend Development

```
$ cd buddy-backend
```

- `.env.example`을 기반으로 `env.development` 파일 생성 및 작성
- Docker로 DB를 구동하기 위해서는 `docker compose -f docker-compose.yml --env-file .env.development up -d`
